Documentation for exim

Exim has a set of documentation released with it. A text file of the main documentation is released as part of the exim tar archive. Additionally postscript and texinfo forms of the documentation are also available in separate tar archives on the ftp sites.

There is also a book, produced by the author - See the book website for further information.

There are 3 main sets of documentation for exim, all of which are also available below in html form.

The main user and configuration manual is the specification document

The Exim Specification - Version 4.3x
The master documentation for exim containing all required detail to install, configure and use exim. Changes to the documentation (normally reflecting changes to the functionality of exim, are now shown in a green font like this segment.
The Exim Filter Specification - Version 4.3x
Additional information on the exim filter language.
